Introduction to AI Agent

什么是

感知环境、自主决策、持续行动——下一代人工智能的核心范式

"

AI Agent（智能体）是一种能够自主感知环境信息， 利用大语言模型或其他 AI 模型进行推理与规划， 并通过调用工具、执行动作来完成复杂目标的软件系统。 它不只是回答问题，而是主动采取行动，直到任务完成。

三大核心能力
🧠

感知 Perception

读取文本、图像、数据库、API 返回等多模态输入，理解当前环境状态。

⚙️

推理 Reasoning

基于大语言模型（LLM）进行链式思考，拆解目标、制定多步骤计划。

🚀

行动 Action

调用外部工具（搜索、代码执行、数据库写入等），与真实世界产生交互。

AutoGPT / Multi-Agent 爆发

自主 Agent 框架涌现，多 Agent 协作系统落地金融、编程等场景。

Agentic AI 全面商用

主流云厂商内置 Agent 能力，MCP 协议统一工具接入标准，Agent 成为新基础设施。

主流 Agent 类型
单体 Agent

单一智能体

一个 LLM 核心 + 工具集，通过循环调用完成任务，结构简单，适合单一垂直场景。

多 Agent

多智能体系统

多个专职 Agent 协作分工，Orchestrator 统筹调度，适合复杂、跨域的长流程任务。

RAG Agent

检索增强 Agent

结合向量数据库实时检索，知识获取与推理解耦，确保回答始终基于最新知识。

具身 Agent

具身智能体

控制机器人或虚拟仿真环境，将感知-决策-执行闭环从数字世界延伸到物理世界。
