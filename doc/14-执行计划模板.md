## 3. 任务分配

使用智能调度命令自动分配：
```bash
team schedule --task "任务描述"
```

系统会自动：
1. 判断任务复杂度
2. 分解为子任务
3. 匹配专业 Agent
4. 生成接口契约
5. 确定执行顺序
